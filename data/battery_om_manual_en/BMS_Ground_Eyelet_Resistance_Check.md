The eyelet, stud face, and return branch need to be read as one mechanical and electrical chain. |
| T3 | Measure resistance from controller ground to main reference ground and then perform a loaded voltage-drop check during wake or ready transition so the joint is judged under current and not only at rest. Record the value at the same operating state used to observe the original symptom. |
| T4 | If the result is poor, remove the eyelet and inspect the eyelet face, serrated washer teeth, stud shoulder, bracket coating, and any oxide film that reduces metal-to-metal contact. A painted land and a flattened serrated washer can create the same drop for different reasons. |
| T5 | Reassemble with the correct washer order and torque only after both mating lands are clean, flat, and centered, then repeat the resistance and voltage-drop proof before the harness support is clipped back into place. This confirms that the improvement belongs to the corrected eyelet interface itself. |
| T6 | Restore the supports and rerun the same wake transition because eyelet faults sometimes return only when the harness branch loads the stud from one side. The repaired ground path has to survive its final pull direction from stud to first clip. |
| T7 | Reinspect the stud stack, washer height, harness lay, and eyelet witness after the repeated proof because the geometry can change once the joint reaches final compression and the loom settles. Post-test hardware condition is part of the diagnosis, not an afterthought. |
| T8 | End the ground review only when the note names the exact fault boundary, such as eyelet oxidation, painted bracket land, missing serrated washer, or loose stud preload, and ties it to the measured resistance and drop before and after correction. The release boundary is a ground path that carries the wake load with stable readings and no renewed shift once the supports are restored. |
