Capture whether the offset stays fixed, grows during wake-up, or narrows after coolant circulation because that trend often separates an electrical bias from a mounting-coupling issue. |
| T2 | Open the BMS temperature-input branch and identify the exact thermistor connector, sensor lead, splice section, harness clip path, mounting point, sensor clip or pad, and board header pin associated with the offset channel before repeated unplugging changes the contact condition. Keep the full route visible from the sensing tip to the board input so a later resistance shift can be assigned to one physical segment and not to the channel generally. |
| T3 | Inspect the sensor connector, seal ring, lead routing, splice section, and the harness path near the bus support and cover flange for heat influence, moisture tracks, conductor strain, partial terminal seating, or a shifted mounting point that could bias the measured temperature. Also compare the suspect sensor body and clip pressure with the neighboring channels so any missing contact pad, loose clip, or changed thermal mass is tied to one local installation difference. |
| T4 | Measure thermistor resistance at the sensor body first and then at the BMS header end, using the same meter, the same ambient condition, and the same lead compensation method, and compare both values with the neighboring reference temperature channel that shares a similar physical environment. The two-point measurement must show whether the resistance shift begins at the thermistor element itself or appears only after the signal passes through the connector, splice, or header path. |
| T5 | If the sensor-side value is correct but the controller-end value is shifted, examine connector contact force, splice integrity, and header-pin condition along the exact measured path. If both values are shifted by the same amount, treat the thermistor element, the mounting contact, or the local thermal coupling as the primary suspect rather than chasing the harness unnecessarily, and compare the suspect mounting face with the two references under the same thermal load. |
| T6 | Review the result against the physical installation by checking whether the sensor body is fully seated at its mounting point, whether thermal paste or clip pressure is missing if applicable, and whether local radiant or coolant-side influence differs from the two reference channels. The fault boundary should separate board-side offset, harness-side resistance increase, and true thermal-coupling error by combining the resistance result with the actual mounting geometry. |
| T7 | Reconnect the branch and close the review only after the offset magnitude, resistance comparison, mounting condition, and confirmed hardware source are recorded together and the boundary is named specifically as sensor bias, connector contact loss, lead resistance increase, splice issue, mounting-coupling problem, or board-side offset. The accepted record has to preserve both the comparison channel IDs and the exact hardware boundary so the next repair does not have to rediscover whether the problem sat in the sensor circuit or in the thermal contact. |
