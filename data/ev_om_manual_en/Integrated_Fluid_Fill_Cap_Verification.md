| T1 | On Aurex FusionDrive-432 EV, start the thermal-system cap review by listing every fill point opened during service, including the battery-loop reservoir cap, the heater-loop fill neck cap, and the onboard-charger cooling cap, because each cap uses a different seal and thread geometry. Keep the review tied to the actual loop serviced at each neck so a correct battery-loop cap is not accepted by mistake on a heater or charger fill point that uses a different depth and vent path. |
| T2 | Inspect each fill neck for cracked threads, damaged seal land, foreign material, coolant crust, and cap identification, making sure the correct cap type is matched to the correct loop before any final pressure or road validation begins. Read the neck lip, locking tabs, and any molded identification mark together so the hardware match is confirmed at the component and not only at the label. |
| T3 | Check the seal condition on every cap, including O-ring cuts, flattened gaskets, missing vent elements, and cross-thread witness marks, then compare those findings with the loop that was serviced so the wrong cap is not silently swapped between neighboring reservoirs. A cap with the right diameter but the wrong vent insert or seal height has to be called out now because it will pass a casual visual check and still fail under thermal expansion. |
| T4 | Tighten each cap to its specified seating condition and verify that the cap reaches the correct stop without rocking, uneven gasket compression, or coolant weeping at the first thermal or pressure-hold event. Confirm that the locking ears land evenly and that the cap cannot be turned farther by hand after it reaches its intended stop, because partial engagement is a common cause of delayed seepage after heat soak. |
| T5 | Run the appropriate pressure-hold or pump cycle for the serviced loops and watch the cap lands, vent paths, overflow hoses, and nearby neck seams for seepage, overflow trace, or cap vent behavior that would show the fill point is not truly sealed under operating condition. Compare the battery loop, heater loop, and charger loop responses separately so one overfilled branch is not mistaken for a bad cap on another reservoir. |
| T6 | If any residue appears, separate cap-seal failure from overfill or runoff by checking the wetting origin at the neck lip, the cap vent path, the overflow hose, and the surrounding hose routing before wiping the area clean. The leak boundary must stop at one cap seal, one vent path, or one neighboring runoff source rather than remain at generic reservoir area level. |
| T7 | Release the vehicle only after the closeout note records the cap identity, seal condition, seating result, operating check outcome, and final dry status for each opened fill point, so later thermal complaints can be traced to a documented final state at every serviced reservoir. The accepted state is not simply caps installed, but each named loop confirmed with the correct cap, a stable seated stop, and no renewed wetting under the same operating condition used for validation. |
