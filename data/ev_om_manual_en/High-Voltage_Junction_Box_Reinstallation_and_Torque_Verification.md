| T1 | Return the bench-tested junction box to Novaris PowerHub-484 AWD only after confirming that the unit matches the original identity, that all internal covers and insulators are present, and that the vehicle-side cable labels and mounting points still correspond to the pre-removal documentation. The reinstall should start from one verified unit identity and one verified vehicle-side branch map so no bench-tested box is mixed with the wrong cable set or mount position. |
| T2 | Position the junction box on its mounting shelf, check bracket alignment and isolator seating, and install the mounting fasteners finger-tight first so the housing can settle squarely before any cable or connector is loaded into position. Read the shelf contact and isolator compression from corner to corner so the housing boundary is established before terminal stacks begin to carry side load. |
| T3 | Reconnect the high-voltage cable lugs in the documented sequence, restoring each stud with its original washer stack and nut order while ensuring the cable lug sits flat on the bus interface with no trapped insulation boot or side-loaded cable angle. Keep the cable route to each stud in its natural support path so clamp load is applied through the interface and not through a twisted cable span. |
| T4 | Torque each terminal stud in sequence, mark the completed torque witness, and compare the final hardware stack height and cable orientation with the pre-removal photos so any incorrect washer order or cable rotation is caught before the cover is closed. Any terminal that settles unexpectedly or changes cable angle during torque has to be reread before the job moves forward. |
| T5 | Reconnect the low-voltage control plugs, feedback connectors, coil connectors, and HVIL branch, checking latch engagement, connector seating, secondary locks, and harness dress so no control-side issue is introduced during the reinstallation. Confirm the branch route from connector body to first clip so the cover and nearby brackets do not push the restored harnesses out of neutral position. |
| T6 | Refit the internal barriers and main cover, verifying that the seal path is clean, the harnesses are not pinched at the cover edge, and the cover fasteners tighten evenly without one corner or pass-through area standing proud. The closure boundary has to prove both barrier restoration and even cover seating on the same housing that was just reconnected. |
| T7 | Restore power in a controlled way and confirm that the junction box remains electrically quiet, mechanically stable, and free of immediate interlock or contactor faults before the torque verification is considered complete. Watch the first ready request, interlock status, and contactor response together so one early electrical instability is not missed after an otherwise correct mechanical reassembly. |
| T8 | Recheck the visible torque witness marks, cover seating, cable support, and connector retention after the first power-up so the final approval reflects both static assembly quality and initial post-install electrical behavior. The release condition is not only a powered box, but a box whose terminal stacks, connector locks, and cover seal remain stable after the first energized cycle. |
| T9 | Close the reinstallation only after the work order includes mount alignment, terminal-stack order, torque sequence, connector verification, cover closure, and the post-install system response for the returned junction box. The final record has to let the next technician see exactly how the unit was remounted, reconnected, and electrically validated in vehicle. |
