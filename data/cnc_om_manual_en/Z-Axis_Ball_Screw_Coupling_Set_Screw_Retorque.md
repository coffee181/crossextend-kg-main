| T2 | Expose the coupling hubs, set screws, shaft flats, keys or clamp features, and the nearby encoder or cable guide while preserving the original hub position and paint marks. Keep upper motor-side and lower screw-side engagement visible together so slip direction can be assigned to one shaft interface. |
| T3 | Measure hub position, set-screw breakout torque, and the amount of lost motion or backlash at the axis, then compare those values with the service target. A set screw that breaks away below target with visible hub walk on one shaft belongs to the coupling boundary even if the axis still moves under light command. |
| T4 | Inspect shaft-flat witness, screw-point seating, hub bore polish, and any key or clamp damage that shows how the coupling moved under load. Compare motor-side marks with screw-side marks so the correction stays on the slipping hub and does not treat both ends as equally suspect without evidence. |
| T5 | Restore shaft indexing, hub seating, and the specified set-screw sequence while holding the hub at the accepted position on both shafts. Confirm that the cable guide and nearby cover do not load the upper hub and that the screw points seat properly on the intended flat or key interface. |
| T6 | Recheck backlash or lost motion after torque, then run a controlled Z reversal or orientation move while the coupling remains observable. If motion improves and witness marks remain stable through the same load change, keep the repair boundary on the corrected set-screw or hub interface rather than chasing servo settings. |
| T7 | Inspect the coupling again after the first loaded cycle for renewed paint break, hub walk, or screw relaxation that only appears once the Z axis transfers real vertical load. This second inspection confirms that the retorque did not only mask the fault during a static setup. |
| T8 | Record the final hub position, set-screw condition, measured backlash or position result, and any supporting hardware corrections in one closeout note. The acceptance evidence must show which screw side slipped, what value was restored, and how the measured motion changed afterward. |
| T9 | Close the work only after repeated Z motion leaves the coupling stable with no renewed set-screw movement and no return of the original lost-motion complaint. Accept the repair when the exact coupling interface has been identified and proven under the same vertical load path that originally exposed the issue. |
