| T1 | For the preventive-maintenance closeout on Novaris MillCenter-900, list the systems that were actually opened or serviced, such as the way-lube reservoir, spindle chiller filter, FRL assembly, coolant basket, cabinet intake filter, and way-cover wipers. The final closeout must follow the real PM scope so later review can tell which subsystems were proven and which were untouched. |
| T2 | Walk the machine exterior from front to rear and then the cabinet interior from upper left to lower right, confirming that every guard, panel, filter cover, drain plug, and fastener removed during PM has been restored. Record any missing clip, loose panel screw, or displaced hose sleeve before functional testing starts, because those small items frequently become the real post-PM failure boundary. |
| T3 | Verify service-state values that can be measured before motion, including lube reservoir level at LR-1, coolant concentration, FRL pressure at PG-A1, chiller setpoint, and cabinet differential-pressure indicator if fitted. Compare those readings with PM target so a low oil level or an underpressurized FRL bowl is caught before it can be blamed on a later motion check. |
| T4 | Trigger a manual lube cycle and inspect pump outlet LP-1, distribution manifold, metering units MU-1 to MU-4, and guide-rail witness points from upstream pump to downstream metering block. If the pump runs but one rail stays dry, keep the PM boundary on that metering branch rather than clearing the whole lubrication system. |
| T5 | Start coolant flow and spindle warm-up, then check the coolant basket lid, pump suction hose, nozzle manifold, spindle chiller inlet and outlet, and any reopened drain connections for leak, rub, or abnormal temperature rise. This couples the fluid systems to real operating demand instead of a static visual pass. |
| T6 | Exercise the door, toolchanger, air-preparation branch, and representative axis motion that cross the serviced areas, paying attention to way-cover movement, door-seal compression, FRL drain closure, and cable support near the work zone. A complete PM closeout has to prove that the serviced machine still moves, seals, and changes tools without the maintenance work creating new interference. |
| T7 | Stop the machine and reinspect the just-tested systems in their final settled state, especially the way-cover wipers, coolant drains, lube fittings, FRL bowl, and cabinet intake panel. Heat soak, vibration, and fluid residue often reveal a weak clip or loose fitting only after the first operating cycle ends. |
| T8 | Run one short combined sequence that includes spindle rotation, axis travel, coolant demand, and one toolchange while the machine remains fully guarded, then compare the machine state with the original PM scope and any pre-PM concern. This final combined proof checks that the independent PM tasks still coexist correctly in production configuration. |
| T9 | Close the PM only after the report names serviced systems, measured pre-motion values, functional checks performed, exact hardware boundaries corrected if any were found, and final guarded machine condition. The release boundary is a machine that holds required service values at LR-1 and PG-A1, completes the combined sequence cleanly, and shows no new leak, rub, loose panel, or unsupported line across serviced areas. |
