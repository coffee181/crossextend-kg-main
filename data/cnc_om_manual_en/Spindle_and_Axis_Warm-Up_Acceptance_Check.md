| T1 | After spindle-cartridge or cooling-line service on Helion MillCenter-800, record the original symptom as high vibration, thermal alarm, taper-face heat, or coolant leak during warm-up, and name the exact components that were disturbed. Note the spindle taper type, toolholder mass, and the previous trouble band, for example 4500 rpm to 6000 rpm, so the acceptance run challenges the same operating window that failed before. |
| T2 | Restore the spindle head to guarded running condition and attach temperature probe TS-SP1 to the front bearing housing plus vibration sensor VB-SP1 to the spindle nose or head casting. Keep chiller inlet hose CH-IN, outlet hose CH-OUT, encoder cable clip, labyrinth seal drain, and fan duct visible together because spindle warm-up faults often move through more than one support path. |
| T3 | Start the approved warm-up sequence from cold start and record spindle current, front-housing temperature rise at TS-SP1, vibration at VB-SP1, and any leak or rub signature at each speed plateau. A pass based only on alarm absence is not enough if the front housing heats too quickly or the vibration trend rises at the same speed that previously triggered the complaint. |
| T4 | At the first higher-speed plateau, inspect the chiller hoses, hose clamps, rotary union drain, fan discharge path, and encoder cable support while the spindle remains running. Coolant mist at the lower drain, a soft upper hose, or a cable branch that touches the fan shroud can explain a returning warm-up problem more precisely than a broad note about spindle instability. |
| T5 | Compare the live readings with the pre-service record or the machine target and decide whether the fault boundary sits on cooling flow, bearing preload, seal drag, fan airflow, or cable interference. If temperature rises faster than expected while vibration stays low, keep the boundary on cooling or seal drag; if vibration climbs while temperature stays normal, keep it on balance, support, or mounting hardware instead. |
| T6 | Correct only the identified hose clamp, bleed point, fan duct, cable clip, or spindle-side support issue, then restart the warm-up from the coldest repeatable state allowed by the method. Record the same measurements in the same order so the second trace can be compared point by point with the first run. |
| T7 | After the spindle reaches the top speed band cleanly, command a tool clamp and unclamp cycle and watch the drawbar response, taper-face blowoff, and encoder stability because some spindle repairs fail again only when pneumatic and rotational loads overlap. This keeps the release boundary on the real operating chain rather than on a no-load spin alone. |
| T8 | Reinspect the front and rear hose clips, fan duct screws, encoder bracket, and drain outlet after the spindle stops because heat soak can reopen a leak or pull a cable onto a rotating surface after the motion event ends. A final dry inspection is required before calling the spindle stable. |
| T9 | Mark the warm-up acceptance complete only when the report ties the serviced parts to measured speed plateaus, temperature rise, vibration trend, and final hardware condition in one traceable closeout. The spindle may go back to service when the original speed band runs without renewed alarm, leak, or abnormal vibration and the cooling branch CH-IN to CH-OUT remains dry and correctly routed. |
