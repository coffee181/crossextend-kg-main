Capture pump cycle interval, oil grade, and the first feed move that exposes the dry-running symptom so the cleaning task stays tied to the real branch. |
| T2 | Expose divider block DY1, inlet manifold, outlet O3, short outlet fitting FT-O3, downstream nylon line LN-Y3, rear guide-truck port, and ball-nut feed line while cleaning only loose oil film. Keep upstream manifold and downstream ports visible together so a healthy inlet pulse is not mistaken for healthy end-of-line delivery. |
| T3 | Trigger the lube cycle and measure inlet pressure at DY1 port P-DY1, then compare delivery timing at outlet O3 and at the actual Y-axis lubrication points. If pressure is present at the block but oil arrives late or not at all downstream, keep the boundary on O3, fitting FT-O3, or outlet line LN-Y3 instead of on the pump reservoir. |
| T4 | Remove only the O3 fitting or the shortest suspect line section and inspect for debris, sealant, collapsed tube, or varnish at the orifice seat. Compare the removed blockage with the neighboring outlet hardware so the repair stays on one metered channel and not on the entire divider block. |
| T5 | Inspect the rear guide-truck port and ball-nut branch separately, then compare which downstream point recovers first after the obstruction is cleared. This split prevents a partial branch recovery from being misread as full Y-axis lubrication recovery when one port is still starved. |
| T6 | Clean or replace the failed outlet hardware, restore the line with the correct ferrule and bend radius, and seat the downstream ports squarely against their fittings. Confirm that the line is supported by its clamp without flattening and that the fitting threads pull fully home against the divider face. |
| T7 | Trigger repeated lubrication cycles until both downstream points show consistent delivery, then run the Y-axis through the feed band that originally produced the dry sound or alarm. The axis is not released on one visible pulse if the rear truck or ball nut starves again after travel changes the line shape. |
| T8 | Reinspect outlet O3, the restored line, the clamp, and both downstream fittings after motion because settling can reopen a weak ferrule or a pinched bend. This second check confirms that the cleaned branch survives the actual machine path instead of only a bench pulse at the block. |
| T9 | Measure or observe end-of-line delivery one more time with the machine fully restored, comparing both the rear-truck branch and the ball-nut branch against the pre-repair state. The release evidence must show that lubrication arrives where the original complaint was seen, not only that the divider face looks clean. |
| T10 | Close the work only after the report names the exact limiting item, such as divider outlet O3, fitting blockage at FT-O3, collapsed line LN-Y3, rear-truck port, or ball-nut branch, and ties it to inlet pressure and end-point delivery. Accept the repair when repeated cycles and repeated Y-axis travel leave both downstream lubrication points wet and the original dry-running symptom absent. |
