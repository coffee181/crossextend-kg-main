| T1 | On Kestrel MillCenter-850, record the auto-lube complaint as no cycle, short cycle, or low-pressure alarm from PS-L1, and note the programmed interval, machine mode, and reservoir level when the failure occurs. Capture whether the issue is visible at the pump, at the feedback switch, or only at the low-pressure alarm. |
| T2 | Expose auto-lube pump LP-1, drive motor or solenoid, suction pickup, reservoir, outlet manifold, pressure switch PS-L1, and cycle feedback device FS-L1 while leaving the downstream network connected for the first observation. Keep the drive side and the pressure side visible together so a running pump can be separated from a pump that actually completes a full lubrication cycle. |
| T3 | Command or wait for a lubrication cycle and record the order of drive start, pressure rise, feedback change, and alarm response. If the drive runs but pressure never rises or the feedback never changes state, keep the next split on suction, pump element, outlet leak, or the feedback device rather than on the timer logic alone. |
| T4 | Measure motor current or solenoid response, then inspect suction pickup, reservoir vent, outlet manifold, and pressure-switch behavior under the same cycle. A pump that sounds normal can still fail to move oil if it has lost prime or if the outlet leaks internally, so the suction and outlet evidence must be read with the cycle timeline. |
| T5 | Compare the observed cycle with a known good cycle length and pressure band for the same machine, then isolate whether the failure belongs to the drive, the pump element, the pressure switch, or the control command. This keeps the repair on one real lube boundary instead of scattering changes across pump, control, and plumbing. |
| T6 | Correct only the identified boundary, restore the outlet manifold and feedback wiring through their original clamps, and confirm that the reservoir vent and pickup path are clear. The repaired branch should return to its normal supported geometry with no pinched line or unsupported cable biasing the pump head. |
| T7 | Repeat multiple lubrication cycles and compare pressure rise, cycle time, and feedback response with the as-found condition. The machine is not released on one successful shot if the second or third cycle still falls short or the low-pressure alarm returns after the reservoir settles. |
| T8 | Reinspect pump head, manifold fittings, feedback device, and the reservoir pickup after the repeated cycles because several auto-lube faults only reappear once temperature and vibration return to the cabinet. This second inspection confirms that the corrected drive and pressure path remain stable in real service. |
| T9 | Record the commanded cycle, measured pressure behavior, feedback response, and the exact corrected boundary in one traceable note. The closeout should let the next technician understand how the cycle was proven from command to pressure to feedback. |
| T10 | Close the work only after the report names the exact failed boundary, such as pump motor, drive solenoid, suction issue, pump element, pressure switch PS-L1, or controller command, and ties it to repeated successful cycles. Accept the repair when the auto-lube system completes the original duty interval with stable pressure and no renewed alarm. |
