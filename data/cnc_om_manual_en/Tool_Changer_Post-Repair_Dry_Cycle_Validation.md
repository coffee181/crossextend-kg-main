Acceptance must be judged in the same closed geometry seen during production, not in an open-bay setup. |
| T3 | Run the first slow dry cycle and timestamp each key event, including carousel index stop, arm pick position, spindle unclamp and clamp response, gripper closure, and pocket return alignment at the serviced station. This timeline provides event-level evidence instead of a single final cycle passed statement. |
| T4 | Extend the proof to include the original failure station plus at least two comparison stations, one adjacent and one remote, then compare stop repeatability, arm clearance, and confirmation-sensor stability across all three. A changer that passes only one pocket does not satisfy release criteria. |
| T5 | Measure the acceptance signal that matches the repaired path, such as clamp cylinder response time, purge-branch pressure at nozzle inlet, proximity trigger distance, pocket seating depth, or Geneva dwell repeatability, and compare each value with the recorded as-found complaint condition. Observational wording is not enough without a measured comparison item. |
| T6 | Reinspect the repaired hardware immediately after the first cycle block for loosened screws, connector walk-out, spring displacement, tubing rub, bracket drift, or fresh witness changes at the corrected boundary. Compare the as-run state with the pre-cycle installation record and log any deviation by exact hardware name. |
| T7 | Execute a second cycle block with all guards and support clips fully secured, then confirm that event timing, station stop quality, and repaired-component stability remain consistent under closed-machine conditions. This second block is the release gate because many changer repairs degrade only after hoses and covers reintroduce production preload. |
| T8 | Close the work only when the note links repaired component, multi-station cycle evidence, measured acceptance value, and post-cycle physical reinspection in one continuous engineering record. The release boundary is a changer that repeats the original sequence window without recurrence and keeps the corrected hardware stable through the full required cycle count. |
