| T1 | On Arcline VTC-630, record the complaint as tool pull-out during face milling, weak unclamp feel, or drawbar alarm on a CAT40 spindle, and note the toolholder style, cutting load, and whether the issue followed recent drawbar service. The intake note must already tie the problem to the drawbar clamp stack and not to a generic spindle fault. |
| T2 | Warm the spindle to the service temperature if required, then expose the drawbar booster cylinder, Belleville stack housing, unclamp valve, pressure transducer port, and the access path to the spindle pull-force test point. Keep the upper actuator side and lower retention side visible in the same inspection window so stroke and clamp force can be judged against the same hardware state. |
| T3 | Install the calibrated clamp-force gauge in a verified CAT40 holder, run the clamp cycle, and record the measured pull force rather than relying on sound or feel. Compare the cold and warmed values if the complaint changes with temperature, because a weak Belleville stack and a short hydraulic stroke do not deteriorate in the same way. |
| T4 | Measure unclamp pressure at the booster cylinder and compare it with the machine target while watching drawbar movement. If the supply pressure is healthy but clamp force remains low, the next split should stay inside the mechanical stack, whereas low pressure with normal stack height pushes the boundary toward the hydraulic side of the circuit. |
| T5 | Inspect the Belleville stack height, washer orientation, release-piston clearance, and actuator linkage from upper cylinder to lower drawbar seat. Pay attention to whether one washer pair is inverted or whether the stack height is short at the lower end, because a single reversed pair can reduce clamp force without leaving obvious damage on the piston or pressure circuit. |
| T6 | Compare measured stack height and washer sequence with the service layout, then isolate the fault boundary only after pressure, stroke, and stack geometry are read together. If pressure reaches the target and release stroke is normal while pull force stays low, keep the fault on the Belleville stack orientation or drawbar hardware instead of blaming the valve or pressure supply. |
| T7 | Rebuild the stack in the correct order, restore the actuator linkage, and repeat the pull-force test with the same toolholder and the same spindle temperature window. Confirm that the stack sits against the lower seat, the linkage travels freely, and the pull-force gauge now follows the corrected washer orientation rather than random warm-up variation. |
| T8 | Run three clamp and unclamp cycles while watching pressure recovery, release stroke, and holder retention, then inspect the actuator and cylinder for fresh oil trace or misalignment. The machine is not released on one good force reading if the drawbar still hangs up on the second or third cycle. |
| T9 | Close the work only after the report names the exact failed boundary, such as an inverted lower Belleville pair, reduced stack height, hydraulic short-stroke, or actuator-linkage misalignment, and ties it to the measured force and pressure results. Accept the repair when clamp force, release stroke, and repeated tool retention all remain stable on the corrected drawbar stack. |
