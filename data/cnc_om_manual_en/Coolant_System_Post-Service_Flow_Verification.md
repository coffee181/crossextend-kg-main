| Time step | O&M sample text |
|---|---|
| T1 | On Kestrel VMC-880, begin the coolant-system validation by naming the serviced branches, such as pump P-1, filter basket SF-1, rotary union RU-1, nozzle manifold MF-2, drain hose DR-4, or routed base hose HB-3, and by restating the original flow or leak complaint. Record coolant concentration, tank level, and the modes that must be proven for release. |
| T2 | Confirm the system is fully reassembled with all guards, clamps, nozzles, covers, filters, and hose supports returned to production position, then verify static readiness before circulation starts. Keep the serviced branches and their nearest supports visible long enough to confirm that the proof cycle is judging the fully assembled coolant path and not an open-service shortcut. |
| T3 | Run standard flood first and record pump current, discharge pressure at PG-C2, nozzle behavior, and leak status at every serviced joint from tank outlet to manifold. Compare the measured values with the original complaint condition so the release note reflects actual recovery instead of a generic pass statement. |
| T4 | If through-spindle coolant or a secondary branch was serviced, switch to that mode and repeat the same inspection chain at the rotary union, branch hose, and delivery point. This split between flood and secondary modes prevents a good flood test from hiding a branch that still fails under the other operating condition. |
| T5 | Measure the acceptance item that matches each serviced boundary, such as flow at nozzle N3, discharge pressure at pump P-1, free drainage at return hose DR-4, or leak-free hold at rotary union RU-1. Movement or pump sound alone is not a valid closeout if the intended delivery or sealing boundary has not been proven. |
| T6 | Reinspect all serviced clamps, clips, sleeves, strain points, and moving hose spans after the first circulation group because pressure and vibration can immediately push a corrected line back into the old stress path. Compare this as-run condition with the as-installed condition recorded before the test. |
| T7 | Run a second proof group with all covers and support clips closed and with the machine in the same operating mode that originally exposed the complaint. This second pass is the real release boundary because several coolant repairs pass while open and fail once hose shape and vibration return to production state. |
| T8 | Record the serviced coolant branches, tested modes, measured acceptance values, and final hardware condition in one traceable closeout note. The report should allow a later technician to identify which branch, which mode, and which measured result were actually used to release the machine. |
| T9 | Close the work only after the report names the final accepted boundaries, such as restored pump suction path, sealed rotary union, stable nozzle-manifold branch, free-return drain, or abrasion-free hose route, and ties them to repeated successful circulation. Accept the coolant system when the original flow or leak complaint no longer appears in the required modes and the serviced hardware remains stable after the full validation sequence. |
