| T1 | On Kestrel VMC-810, begin the Y-axis post-repair cutting audit by naming the repaired components and the exact feature that originally exposed the problem, such as bore mismatch, wall-step error, or reversal mark. Record the verification program, feed rate, depth of cut, and the required feature tolerance before the machine starts. |
| T2 | Confirm the repaired Y-axis hardware is fully restored with covers, clamps, lubrication lines, switch brackets, and cable guides in production position, then verify static readiness before cutting. Keep the repaired boundary and its nearest support path visible long enough to confirm that the machine is being judged in its final assembled geometry. |
| T3 | Run the verification cut that reproduces the original Y-axis load path and motion window, while monitoring servo load, feed behavior, and any abnormal sound or vibration. The audit must use the same style of motion that previously revealed the fault, not a simplified dry run that avoids the original load. |
| T4 | Measure the finished feature at the locations that correspond to the original complaint, then compare the result with target and with the pre-repair condition if available. A repaired axis that moves quietly but still misses the feature tolerance has not actually crossed the release boundary. |
| T5 | Reinspect the repaired Y-axis hardware after the cut, checking screws, clamps, ferrules, cable branches, lubrication points, and way-cover contact for movement or leakage that appeared only under real cutting load. Compare this as-run condition with the as-installed condition recorded before the cycle. |
| T6 | Repeat the same cut or a shorter confirming pass if required and verify that the feature result and the hardware condition remain stable on a second loaded cycle. The machine should not be released on a single good part if the hardware shifts or the dimension drifts immediately afterward. |
| T7 | Record the exact motion window, cut condition, measured feature result, and the final state of the repaired boundary in one closeout chain. This keeps the release evidence usable for later troubleshooting instead of reducing it to an untraceable pass note. |
| T8 | Close the work only after the report links repaired Y-axis hardware, verification cut, measured outcome, and post-cut physical inspection into one traceable record. Accept the repair when the same cutting condition that once produced the problem now finishes within tolerance and leaves the repaired hardware stable and dry. |
