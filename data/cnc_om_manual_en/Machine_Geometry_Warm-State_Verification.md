| T2 | Bring the machine to the required warm state by running the spindle at the prescribed speed band and traversing X, Y, and Z through the specified travel window, then stop with the table centered and the column at the same Z height used in the baseline check. Keep the spindle nose, table surface, front leveling pads, and rear column support area accessible because thermal geometry drift often links a measurement error to one support or growth path. |
| T3 | Install the granite square, BT40 test bar, and dial indicator or spindle probe exactly as defined in the service method, then measure the target relation at the front-left and rear-right table positions. Record both the measured deviation and spindle-housing temperature at sensor TS-SP1 when reading is taken, because geometry results collected at mismatched temperature are not comparable. |
| T4 | Compare the warm-state result against the accepted baseline and against the cold reading only after confirming that spindle speed history, chiller flow, and way-lube cycle count match the comparison basis. If the deviation appears only after the spindle housing climbs into the higher temperature band, keep the boundary on thermal growth, leveling support, or ballscrew-support preload rather than on probe setup alone. |
| T5 | Inspect front-left and rear-right leveling studs, Y-axis rear support bracket, spindle chiller inlet and outlet flow at CH-IN and CH-OUT, and X-axis support bearing housings for evidence that supports measured drift. A wet chiller fitting, shifted leveling witness, or a support block that heats more than the opposite side gives a clearer boundary than a generic note about warm geometry error. |
| T6 | Correct only the identified boundary by restoring leveling preload, tightening the support hardware, clearing chiller restriction, or resetting the support bracket, then repeat the same warm-up and the same measurement positions. Confirm that the machine returns to the identical table and spindle locations so the new reading can be compared directly with the original drift value. |
| T7 | Reinstall any removed covers and run one more shortened thermal cycle because sheet-metal preload and hose routing can move a marginal support point enough to change the warm reading. This second pass proves that the accepted geometry survives the final front-to-rear machine configuration instead of an open-service setup. |
| T8 | Return the machine to production only when the record names the exact thermal boundary, such as front-left leveling stud, Y-axis rear support bracket, spindle chiller branch CH-IN to CH-OUT, or support-bearing stack, and lists the before and after geometry values under the same warm condition. The release boundary is reached when the warm-state geometry stays within target at both measured table positions and no renewed thermal drift appears on the repeat cycle. |
