| T1 | On Novaris VMC-600, record the hose-abrasion complaint at the machine base and note whether the wear is preventive only or already causing leakage, then identify the exact hose span and mating edge before anything is moved. Capture current pressure, the routing from hydraulic manifold HM-3 to base feedthrough BF-1, and the first visible rub mark. |
| T2 | Expose the full hose span, the base edge, P-clamp PC-7, abrasion sleeve, nearby moving hardware, and both end fittings while preserving the as-found routing. Keep the upper fitting, lower rub point, and adjacent clamp visible together so the cause can be assigned to one contact pair instead of to the entire branch. |
| T3 | Inspect the outer cover, reinforcement layer, sleeve condition, clamp position, and edge finish to decide whether only the jacket is worn or whether the hose wall has already been compromised. Compare the damaged span with a healthy routed span so the diagnosis lands on one support geometry and not on a generic hose age assumption. |
| T4 | Measure clearance and hose movement under pressure or vibration at the original operating state, then compare the hose position with and without clamp support. If the branch clears the edge only when the clamp is held by hand, keep the fault on clamp location or support geometry rather than on the hose fitting itself. |
| T5 | Replace or reroute the hose as required, restore the abrasion sleeve and edge protection, and set the clamp so the hose keeps the intended bend radius from upper fitting to lower base passage. The corrected branch should clear the original edge through the full machine vibration path without twisting the end fittings. |
| T6 | Pressure-test the repaired or rerouted branch and inspect both end fittings, clamp PC-7, and the corrected rub point for new stress. The release boundary is not only a dry hose but a hose whose support geometry no longer drives it back into the original abrasion interface. |
| T7 | Close the work only after the report names both the worn hose span and the exact mating edge, clamp, or bracket that created the abrasion, and ties that boundary to pressure-test and clearance results. Accept the repair when the hose remains dry and clear of the original rub point under the same pressure and vibration condition that produced the wear. |
